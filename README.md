# OpenClaw 小白指南

🚀 从新手到精通，采集网络世界各种好玩的 OpenClaw 技巧

## 📝 网站地址

在线访问：https://你的用户名.github.io/openclaw-guide

## 🛠️ 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建
npm run docs:build
```

## 📂 目录结构

```
docs/
├── guide/          # 新手指南
├── tips/           # 进阶技巧
├── examples/       # 实战案例
└── .vitepress/     # 配置文件
```
