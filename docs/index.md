## 🎯 为什么选择 OpenClaw？

OpenClaw 是一款开源的本地 AI 助手框架，与传统聊天机器人的本质区别在于——**它能真正执行任务**。

| 传统 AI 聊天工具 | OpenClaw |
|----------------|----------|
| 只能回答问题 | 能读写文件、执行命令 |
| 数据上传云端 | 本地运行，隐私可控 |
| 被动等待指令 | 主动定时执行任务 |
| 功能固定 | 无限扩展技能 |
