# 浏览器自动化

使用 OpenClaw 控制浏览器完成自动化任务。

## 🚀 基础用法

### 打开网页
```
帮我打开 https://www.example.com
```

### 截图
```
截图当前页面
```

### 点击元素
```
点击页面上的"登录"按钮
```

## 🎯 实战案例：自动填表

```
打开 https://forms.example.com
在姓名框输入"张三"
在手机号框输入"13800138000"
点击提交按钮
截图确认
```

## 💡 高级技巧

### 批量数据采集
```
访问 https://news.example.com
提取所有文章标题和链接
保存到Excel文件
```

### 定时监控
配合 cron 功能，可以定时检查网页变化。
