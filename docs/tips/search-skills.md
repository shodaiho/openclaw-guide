# 搜索技巧大全

OpenClaw 的搜索功能非常强大，掌握这些技巧让你效率翻倍。

## 🔍 基础搜索

### 普通网页搜索
```
帮我搜索"OpenClaw 最新版本"
```

### 限制时间范围
```
搜索最近一周的 AI 新闻
```

### 指定网站搜索
```
在 GitHub 上搜索 OpenClaw 相关项目
```

## 🎯 高级技巧

### 1. 多源对比搜索
```
帮我对比搜索：
1. OpenClaw 官方文档
2. 社区使用教程
3. 实际案例分享
```

### 2. 结构化输出
```
搜索 2024 年前端框架趋势，用表格对比：
- 框架名称
- GitHub Stars
- 主要特点
- 适用场景
```

### 3. 深度挖掘
```
搜索 OpenClaw 的 Skill 开发教程，要求：
- 包含完整代码示例
- 有实际运行截图
- 标注版本兼容性
```
