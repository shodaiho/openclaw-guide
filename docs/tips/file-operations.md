# 文件操作妙招

OpenClaw 可以帮你高效处理各种文件任务。

## 📁 基础操作

### 批量重命名
```
把文件夹里所有的 .txt 文件重命名为 .md
```

### 批量转换
```
把文件夹里的所有图片压缩到 800px 宽
```

### 文件搜索
```
找出项目中所有包含 "TODO" 的文件
```

## 🎯 实战案例

### 案例1：整理下载文件夹
```
分析 ~/Downloads 目录
按文件类型创建子文件夹（图片/文档/压缩包）
把文件自动分类移动
```

### 案例2：批量处理 Excel
```
读取 data.xlsx
筛选出销售额大于 10000 的行
生成汇总报告
保存为 report.xlsx
```

### 案例3：代码统计
```
统计 src 目录下的代码行数
按文件类型分组
生成可视化图表
```
