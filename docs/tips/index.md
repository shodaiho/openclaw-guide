## 🎯 技巧分类

### 📁 文件处理技巧

#### 1. 批量文档总结
```
帮我总结 /documents 文件夹里所有 PDF 文件的核心内容，
生成一份汇总报告，包含：
- 每个文档的标题和摘要
- 关键信息提取
- 按主题分类整理
```

#### 2. 代码批量注释
```
读取 src/ 目录下的所有 JavaScript 文件，
为每个函数添加 JSDoc 注释，

```

#### 3. 图片批量处理
```
把 /images 文件夹里的所有图片压缩到 800px 宽，
保持比例，保存为 webp 格式。
```

#### 4. 数据格式转换
```
把 data.xlsx 转换成 JSON 格式，
并验证数据完整性，
生成转换报告。
```

## 💡 使用建议

### 提示词优化原则

1. **具体明确**
   - ❌ "帮我处理文件"
   - ✅ "把 /docs 下的所有 Markdown 文件转换成 PDF"

2. **分步骤执行**
   - 复杂任务拆分成多个小步骤
   - 每步确认后再继续

3. **提供上下文**
   - 说明文件格式、编码方式
   - 指定输出要求

4. **验证结果**
   - 重要操作前先预览
   - 生成后检查结果
