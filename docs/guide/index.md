## 💡 常见问题

**Q: OpenClaw 收费吗？**
A: OpenClaw 本身是开源免费的，但调用 AI 模型（如 GPT-4）需要 API 费用。

**Q: 需要会编程吗？**
A: 基础使用不需要，但开发高级功能需要一点编程基础。

**Q: 支持哪些平台？**
A: 目前支持钉钉、飞书、Telegram 等，还可以集成到 VS Code。
