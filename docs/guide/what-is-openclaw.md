# 什么是 OpenClaw

OpenClaw 是一个**开源的 AI 助手框架**，让你可以：

- 🤖 搭建属于自己的 AI 助手
- 🔧 开发各种自动化技能（Skills）
- 💬 接入微信、钉钉、飞书等平台
- 🧠 拥有长期记忆能力

## 🎯 核心理念

OpenClaw 的核心理念是：**AI 是你的助手，不是替代品**。

它通过以下方式实现：

1. **SOUL.md** - 定义 AI 的性格、行为准则
2. **AGENTS.md** - 配置工作区和工具
3. **MEMORY.md** - 长期记忆存储
4. **Skills** - 各种实用功能插件

## 🚀 能做什么？

| 场景 | 示例 |
|------|------|
| **信息查询** | 查天气、搜新闻、找资料 |
| **文件处理** | 批量重命名、格式转换、数据分析 |
| **开发辅助** | 代码审查、自动生成文档、Bug 修复 |
| **生活助手** | 定时提醒、行程安排、记账 |
| **内容创作** | 写文章、生成图片、制作视频 |
