# 投稿指南

欢迎分享你的 OpenClaw 使用技巧！

## 📝 投稿方式

### 方式一：GitHub PR（推荐）
1. Fork 本仓库
2. 在 `docs/tips/` 或 `docs/examples/` 创建 Markdown 文件
3. 提交 Pull Request

### 方式二：钉钉投稿
添加好友 **OpenClaw助手**，直接发送你的技巧。

## 🎯 内容要求

### 技巧类文章结构
```markdown
# 技巧标题

## 场景
描述这个技巧能解决什么问题

## 操作步骤
1. 第一步
2. 第二步
3. 第三步

## 完整示例
```
实际可用的 Prompt 示例
```

## 注意事项
- 注意点1
- 注意点2
```

### 案例类文章结构
```markdown
# 案例标题

## 背景
描述应用场景

## 解决方案
详细步骤

## 效果展示
截图或输出示例

## 总结
经验心得
```

## ✅ 审核标准
- [ ] 原创或已获得授权
- [ ] 步骤清晰可复现
- [ ] 有实际使用价值
- [ ] 符合社区规范

## 🎁 投稿奖励
- 优秀投稿者列入贡献者名单
- 定期评选"最佳技巧"送出周边
